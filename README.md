Sveltekit version of Letsplit.

Used to simplify debt resolving in a friend group by reducing number of payments.

## How to use

You enter the people involved in the group.
You state what each person has payed.
The application shows total amount and amount per person.
The application calculates what each person owes and is owed.

The application shows who should send what amount to resolve what each person owes and is owed.

**Note:** This application assumes everyone contributes an equal amount of money.
